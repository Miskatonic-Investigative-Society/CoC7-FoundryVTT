# Importateur d'acteurs
Vous pouvez utiliser l'importateur d'acteurs pour importer plusieurs blocs de PNJ/Créatures depuis des aventures en utilisant l'exportateur JSON de [The Dhole's House](https://www.dholeshouse.org/).
Pour ouvrir l'importateur d'acteurs, ouvrez le *Répertoire des acteurs* et cliquez sur *Importateur d'acteurs* en bas de la barre latérale, ou depuis une scène active, puis cliquez sur *Importateur d'acteurs*.

# Aperçu
Si c'est votre première fois, il est recommandé de lire également les sections suivantes :
- Personnage non-joueur (PNJ) / Créature
- Importateur JSON d'acteurs de The Dhole's House

# Personnage non-joueur (PNJ) / Créature
![](../../assets/manual/importer/importer.webp)
1. Sélectionnez PNJ ou Créature
2. Le système doit-il convertir le bloc de personnage depuis une édition antérieure vers la 7ᵉ ?
3. Sélectionnez les langues du bloc de personnage
4. Lors de l'ajout de compétences, objets, sorts et armes, le système peut tenter de trouver des éléments dans votre monde portant le même nom. Vous pouvez sélectionner l'ordre de recherche des sources :
   _Objets_ : Depuis votre répertoire d'objets
   _Monde_ : Depuis les compendiums de votre monde
   _Modules_ : Depuis les compendiums de vos modules
   _Système_ : Depuis les compendiums système fournis avec ce système
5. Un exemple de mise en page est donné ici. Vous pouvez le copier dans votre presse-papiers si vous souhaitez l'éditer ou coller le texte depuis une aventure.
Cliquez sur *Importer* pour créer un acteur dans le *Répertoire des acteurs*, dans le dossier *Personnages importés*. Tout texte non compris sera stocké dans les *Notes du Gardien*.

# Importateur JSON d'acteurs de The Dhole's House
![](../../assets/manual/importer/actor.webp)
1. Importateur JSON d'acteurs de The Dhole's House
2. Lors de l'ajout de compétences, objets, sorts et armes, le système peut tenter de trouver des éléments dans votre monde portant le même nom. Vous pouvez sélectionner l'ordre de recherche des sources :
   _Objets_ : Depuis votre répertoire d'objets
   _Monde_ : Depuis les compendiums de votre monde
   _Modules_ : Depuis les compendiums de vos modules
   _Système_ : Depuis les compendiums système fournis avec ce système
Parcourez vos fichiers pour sélectionner votre fichier JSON. Une fois sélectionné, le nom et l'image seront affichés. Cliquez sur *Importer* pour créer l'acteur dans le *Répertoire des acteurs*, dans le dossier *Personnages importés*.
Par défaut, l'image sera stockée dans un dossier nommé *dhole-image* dans votre monde. Vous pouvez modifier cela en cliquant sur l'onglet *Paramètres du jeu*, puis sous l'en-tête *Paramètres du jeu*, cliquez sur *Configurer les paramètres*, puis sur *Paramètres système*.
